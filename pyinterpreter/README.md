# Interpreter 
Интерпретатор  
## Запуск тестов (необходим Pytest):
```
pytest
```
## Построение отчета о покрытии
```
coverage run -m pytest tests/
coverage html
```
Отчет о покрытии искать в __htmlcov/index.html__
